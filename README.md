# AnyMMO Fishnet Addon

## Introduction

This is the FishNet MMO addon for AnyRPG.

It adds the following features and functionality:
* FishNet compatible character and mount prefabs
* Template content that includes FishNet NPCs and Interactables
* A FishNet MMO demo game to demonstrate how it all works with AnyRPG

## Installation

This addon can be installed via the Welcome Window in AnyRPG.

https://github.com/AnyRPG/AnyRPGCore
